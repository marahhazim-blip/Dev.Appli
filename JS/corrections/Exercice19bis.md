## Exercice sur les fonctions

### Objectifs 

1. Donner une fonction qui permet de donner l'heure avec 2 parametres (heure, minute), celle-ci retournera une chaine de caractere.
Le résultat sera sous la forme : "14h00" par exemple. 
Donner une valeur par défaut aux parametres pour qu'elle affiche "12h00" si aucune valeur n'est transmise.

```javascript
function donnerHeure(heure, minute){
    // Instruction
}

alert(donnerHeure(14, 00)) // 14h00 
alert(donnerHeure()) // 12h00 
```

2. Donner une fonction permettant d'afficher si un nombre est pair ou impair.
La fonction enverra true si le nombre est pair.
