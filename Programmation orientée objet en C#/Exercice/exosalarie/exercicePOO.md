# Exerccie Salarie

Creer une classe Salarie avev les attributs suivant :

- un matricule => valeur par defaut "000"
- un service => valeur par defaut "service par défaut"
- un nom => valeur par defaut "nom par default"
- une categorie => valeur par defaut "categorie par defaut"
- un salaire => valeur par default 2000

On ajoutera autant de constructeur que necessaire

Une méthode pour obtenir les details d'un employe (toutes ces informations sous forme de chaine de caracteres)

Dans le programme principal vous creez plusieurs salarie et vous allez parcourir chacun de ces salaries pour afficher ces details.
