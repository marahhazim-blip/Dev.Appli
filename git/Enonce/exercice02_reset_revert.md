# Exercice 02 - reset & revert

## Taches

1. Créer un nouveau repository Git
2. Ajouter un fichier1 et le commiter (commit 1)
3. Ajouter un fichier2 et le commiter (commit 2)
4. Ajouter un fichier3 et le commiter (commit 3)
5. Modifier le fichier3 et le commiter (commit 4)
6. Observer l’historique (on doit voir les 4 commits)
7. Afficher la différence entre le commit 4 et le commit 1.
8. Annuler le commit 3 (le fichier 2 doit disparaitre)
9. Réinitialiser jusqu'au commit 1 mais en conservant les modifications effectué.
10. Effectuer un commit
11. Observer l’historique (on doit voir deux commits)
